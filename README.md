# Jira Help Desk Ticketing Lab

A Jira help desk ticketing lab simulating a real IT support queue with ticket lifecycle, workflows, and common IT scenarios.

## Overview

This repository contains a case study lab that demonstrates professional IT support operations using Jira as the ticketing system. It includes real-world scenarios, incident responses, and best practices for IT helpdesk management.
